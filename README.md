# ProyectoRetroBattle

Juego simple estilo piedra, papel o tijera con vidas y gráficos ASCII. El proyecto está pensado para que varios compañeros colaboren fácilmente en cada clase.

Estructura:

- `src/Main.java`
- `src/Juego.java`
- `src/AsciiArt.java`
- `src/Animaciones.java`
- `src/Jugador.java`
- `src/CPU.java`
- `src/Utilidades.java`

Cómo ejecutar:

1. Abre el proyecto en tu IDE de Java.
2. Compila los archivos en `src/`.
3. Ejecuta `Main`.

Descripción:

- `Main` inicia la aplicación.
- `Juego` controla el bucle principal, las vidas y el flujo de la partida.
- `AsciiArt` contiene las imágenes ASCII y la presentación visual.
- `Animaciones` contiene efectos de texto sencillos.
- `Jugador` modela al jugador humano.
- `CPU` modela al oponente de la máquina.
- `Utilidades` gestiona entradas, conversiones y pequeñas ayudas.
